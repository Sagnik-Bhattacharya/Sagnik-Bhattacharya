#### [🔗 NeuroDB — Custom Database Engine](https://github.com/Sagnik-Bhattacharya/NeuroDB)
A low-level, file-based document database built from scratch to explore database storage engine internals.
* **Architecture:** Features a Write-Ahead Log (WAL) for durability, ACID transaction boundaries, and a crash-recovery mechanism to prevent data corruption.
* **Tech Stack:** Python, Low-level OS I/O operations, File Locking mechanisms.

#### [🔗 SentinelAI — Secure AI Execution Framework](https://github.com/Sagnik-Bhattacharya/SentinelAI) *(Implicit Repo)*
A secure runtime environment built for executing autonomous AI actions within strict, permissioned guardrails.
* **Architecture:** Leverages deterministic JSON action plans parsed through a strict validation engine before execution occurs within safe, containerized sandbox isolation zones.
* **Tech Stack:** Python, Docker SDK, Pydantic Schema Validation.

#### [🔗 SmartCLI — Intent-Driven Shell Assistant](https://github.com/Sagnik-Bhattacharya/smartcli)
A context-aware terminal utility that safely handles system management by matching raw natural language intents to actionable shell streams.
* **Architecture:** Integrates multi-step planning loops, persistent conversational memory blocks, a strict risk validation gate, and automatic error-handling recovery processes.
* **Tech Stack:** Python, Ollama, Rich UI library, OS Native Execution Layers.

#### [🔗 DataBrain & MLMate — AutoML Suite](https://github.com/Sagnik-Bhattacharya/databrain)
An advanced command-line tool suite that fully automates the machine learning lifecycle—from raw profiling up to automated inference deployment.
* **Architecture:** Generates complete data-cleaning routines, dynamically tests classification/regression structures via Scikit-Learn, evaluates hyperparameter spaces via Optuna, and provides black-box model explanations using SHAP.
* **Tech Stack:** Python, Scikit-Learn, Pandas, SHAP, Optuna, Click.
